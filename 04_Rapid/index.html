Week 4:

Rapid Prototyping

For the week 4 assignment, I had two things I had in mind to do. The first is to create a kinetic art piece that simulated an airplane flying. The second was to create some sort of flap/aileron.

(This weekend, my 3D printer arrived so I wanted to mess around with it.)

The first thing I decided to do was program the code for the motor and complete the circuitry. For the kinetic art piece, I had a potentiometer that controlled the motor's speed of rotation. The program was straightforward.


As you can see from the code, I made A0 and A4, originally input pins, into output pins. I then made A0 into ground, and A4 into 3.3V. Knowing that the potentiometer would output to A2, I defined a constant as the A2 output. This output was then scaled down from 0-1023 to 0-255. I then set the motor rotation speed to that value.

After printing everything and using hot glue to secure the parts, I placed the plane onto the motor. The result was not fantastic, but everything worked.






My next project was to attempt to make flaps. The first step was to code the program. This program was a little more complex than the previous one.


Since this code requires directional changes, I had to define it; I did this with the constant B. The speed the motor turns is now controlled by constant A. From here, many things are pretty similar-A2 is still giving the value of the potentiometer, and I am still mapping that value from a 0-1023 scale to 0-255. The main difference is that there is now an if, then statement. That statement makes half the potentiometer values turn the motor one way, and the other half make the motor turn in the opposite direction. (circuit for motor is still the same.)


Building this flap was surprisingly difficult, and I ran into some problems on the way. Here is my original prototype:


However, just as I started to build it, I found that I did not have a hinge. This meant my first design would move linearly rather than rotationally. After brainstorming, I decided to make a hinge with a toothpick. Then I realized I should connect the flap with a long straw; the straw would rotate the flap. After the prototype was complete, the building was easy. I made use of styrofoam for ease of design, and toothpicks were stuck into them. Some hot glue was used to secure the motor to the styrofoam circle.
